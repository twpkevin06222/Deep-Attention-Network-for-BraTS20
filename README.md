# Deep Supervised Attention Networks for Pixel-wise Brain Tumour Segmentation 

## Dataset 
The aim of the project was to solve the Multimodal Brain Tumour Segmentation Challenge 2020 ([BraTS20](https://www.med.upenn.edu/cbica/brats2020/data.html)). Training and validatin dataset can be obtain upon request at the official website (subjected to registration).
